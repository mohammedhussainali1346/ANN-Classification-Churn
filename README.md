# 🏦 Bank Customer Churn Prediction using ANN

This project predicts whether a bank customer is likely to churn using an Artificial Neural Network (ANN).  
The model is deployed using Streamlit for an interactive web-based interface.

---

## 📌 Project Overview

Customer churn prediction helps banks identify customers who are likely to leave the bank.  
This project uses:

- Artificial Neural Networks (TensorFlow / Keras)
- Data preprocessing (Encoding + Scaling)
- Streamlit for deployment

---

## 🚀 Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- TensorFlow / Keras
- Streamlit

---

## 📊 Features Used

- Credit Score
- Geography
- Gender
- Age
- Tenure
- Balance
- Number of Products
- Has Credit Card
- Is Active Member
- Estimated Salary

---

## 🧠 Model Architecture

- Input Layer
- Hidden Layers (Dense + ReLU)
- Output Layer (Sigmoid activation)

The output provides a probability score indicating the likelihood of customer churn.
